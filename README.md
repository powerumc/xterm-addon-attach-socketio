xterm-addon-attach-socketio
------------------------------
It's used [socket.io](https://github.com/socketio/socket.io) server and client without origin WebSocket.

## Installation

```bash
npm install xterm-addon-attach-socketio
```

## Demo

### Server
```bash
npm i

tsc -p demo
node demo/server.js
```

### Client
```bash
open demo/index.html
```
